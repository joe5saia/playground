## Top Recommendation: Astro

**Why Astro?**
- ✅ Zero JavaScript by default (addressing your #1 complaint about WordPress)
- ✅ Exceptional SEO performance
- ✅ Built-in blog support with Markdown
- ✅ Easy to update - just edit .md files
- ✅ Perfect for content-heavy sites
- ✅ 10-100x faster than WordPress

**Content Management Options:**
1. **Decap CMS** - Free admin UI for non-technical users
2. **Markdown Files** - Even simpler, edit directly in GitHub or VS Code
